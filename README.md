# 🚀 Animated Portfolio Website

A stunning, feature-rich portfolio website built with React, TypeScript, Tailwind CSS, and Framer Motion.

## ✨ Features

### 🎨 Animations & Effects

1. **Loading Screen** - Animated spinner with fade transitions
2. **Advanced Custom Cursor** - Shape-shifting cursor that changes based on hover context (text, buttons, default), with trailing dots and glow effects (desktop only)
3. **Dynamic Background** - Constantly morphing gradient blobs, floating particles, animated lines, and mesh gradients that never stop moving
4. **Scroll Animations** - Elements fade and slide in as you scroll
5. **Hover Effects** - Cards, buttons, and icons respond to hover
6. **Gradient Animations** - Rotating and pulsing gradient backgrounds
7. **Typing Animation** - Rotating text showing different roles
8. **Stagger Animations** - Sequential appearance of elements
9. **3D Transformations** - Rotating and scaling effects
10. **Smooth Page Transitions** - Animated navigation between sections
11. **Scroll to Top Button** - Animated button that appears on scroll with rotation effect
12. **Cursor Variants** - Cursor changes size, color, and shape based on what you're hovering over
13. **Background Morphing** - Multiple gradient blobs that constantly move, scale, and rotate
14. **Floating Particles** - 20+ particles that float around the screen
15. **Animated SVG Lines** - Path animations that draw and fade continuously

### 🌓 Theme Toggle

- Light/Dark mode switcher with smooth transitions
- Theme preference saved to localStorage
- Animated sun/moon icon transition
- All components adapt to theme changes

### 📱 Sections

1. **Hero** - Eye-catching introduction with profile, typing animation, and social links
2. **About** - Personal information with animated feature cards
3. **Skills** - Tech stack with skill level bars and icon animations
4. **Projects** - Portfolio projects with hover effects and tags
5. **Experience** - Timeline view of work history
6. **Contact** - Contact form and information cards

## 🎨 Customization

To customize the portfolio for your own use:

1. **Personal Information**: Edit name, role, description in `Hero.tsx`
2. **Skills**: Update skills array in `Skills.tsx`
3. **Projects**: Modify projects data in `Projects.tsx`
4. **Experience**: Update work history in `Experience.tsx`
5. **Contact**: Change contact info in `Contact.tsx`
6. **Colors**: Adjust Tailwind classes throughout components
7. **Social Links**: Update URLs in `Hero.tsx`
